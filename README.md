# Igreja Seguidores · Site Institucional

Site institucional da Igreja Seguidores, desenvolvido voluntariamente com foco em acessibilidade, performance e experiência mobile-first.

🌐 **[igrejaseguidores.com.br](https://www.igrejaseguidores.com.br)**

---

## Tecnologias

- **React** — biblioteca principal de UI
- **TypeScript** — tipagem estática
- **Vite** — bundler e dev server
- **Google Sheets API** — integração do formulário de contato com planilha

---

## Funcionalidades

- **Home** — apresentação da igreja
- **Sobre** — história e missão da comunidade
- **Galeria** — fotos dos cultos e eventos da igreja
- **Contato** — formulário integrado com Google Sheets via API

---

## Abordagem

O projeto foi desenvolvido com **mobile-first**, garantindo uma boa experiência em dispositivos móveis antes de escalar para telas maiores.

---

## Rodando localmente

```bash
# Instalar dependências
npm install

# Iniciar em modo de desenvolvimento
npm run dev

# Gerar build de produção
npm run build
```
